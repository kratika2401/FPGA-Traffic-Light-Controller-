PROJECT TITLE

FPGA-Based Traffic Light Controller

OBJECTIVE

To design and verify a traffic light controller using Verilog HDL and Finite State Machine (FSM) methodology.

FEATURES

- Traffic light sequencing
- Pedestrian crossing request
- Emergency mode
- Night flashing mode
- Debounced inputs

DIGITAL DESIGN CONCEPTS USED

- FSM
- Sequential Logic
- Combinational Logic
- Timers
- Clock Enable Generator
- Input Synchronization

FSM STATES

S_NS_G
S_NS_Y
S_ALL_RED1
S_EW_G
S_EW_Y
S_ALL_RED2
S_PED_WALK
S_EMERG_ALL_RED
S_NIGHT

SIMULATION RESULT

The design was compiled successfully using Icarus Verilog.

Simulation was executed successfully using:

vvp traffic_sim

Output waveform file:

traffic.vcd

CONCLUSION

The traffic controller was successfully designed, simulated, and verified using Verilog HDL. The design demonstrates practical FPGA and VLSI concepts such as FSM design, timing control, input conditioning, and verification methodology.
